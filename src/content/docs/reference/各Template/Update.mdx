# UPDATE
- 各テーブルのIDは必須
- それ以外は任意で、jsonに項目があるもののみ書き換えられる

## update_shop
```json
{
    "id": 1,
    "name": "Shop Name",
    "mail": "shop@domain.jp",
    "phone": "XXXX-XX-XXXX",
    "address": "Tokyo"
}
```
